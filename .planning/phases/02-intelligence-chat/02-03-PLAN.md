## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| NewsAPI article data → DOM | External article titles and URLs rendered as HTML; XSS risk if titles contain script tags |
| Article URLs → browser navigation | External URLs opened in new tab |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02-09 | Tampering | NewsCard article title rendering | mitigate | React JSX renders text content as text nodes (no `dangerouslySetInnerHTML`); article titles are rendered as `{article.title}` inside JSX — React escapes HTML automatically |
| T-02-10 | Information Disclosure | Article URLs in `<a href>` | mitigate | `rel="noopener noreferrer"` on all article links prevents opener context leakage and tab-napping |
| T-02-11 | Tampering | Rec badge colour from h.rec field | accept | REC_STYLES lookup: only "BUY", "HOLD", "SELL" keys have entries; unknown values render no badge (h.rec falsy check guards rendering); no eval or dynamic class injection |
| T-02-SC | Tampering | npm build (no new packages) | accept | No new npm packages added in this plan; only JSX/CSS changes to existing React project; no postinstall risk |
</threat_model>
